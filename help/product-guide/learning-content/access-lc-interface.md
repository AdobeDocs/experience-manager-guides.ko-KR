# 학습 및 교육 콘텐츠 인터페이스 액세스

학습 및 교육 컨텐츠 인터페이스에 액세스하려면 다음 단계를 수행하십시오.

1. AEM 인스턴스에 로그인.
2. AEM 탐색 페이지에서 **안내서**&#x200B;를 선택합니다.
3. 이제 **Experience Manager Guides 홈 페이지**&#x200B;에 있습니다. 탐색 전환기를 사용하여 편집기 보기로 전환합니다.

   ![](assets/aem-navigation-switcher.png){width="300" align="left"}

4. **저장소** 패널을 열고 과정을 만들 폴더로 이동한 다음 **옵션** 메뉴에서 **새로 만들기 > 과정**&#x200B;을 선택합니다.

   ![](assets/create-new-course.png)
