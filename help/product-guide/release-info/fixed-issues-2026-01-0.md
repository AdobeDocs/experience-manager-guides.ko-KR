# 2026.01.0 릴리스의 문제가 해결되었습니다.

이 문서에서는 Adobe Experience Manager Guides as a Cloud Service의 2026.01.0 릴리스의 다양한 영역에서 수정된 버그를 다룹니다.

새로운 기능 및 개선 사항에 대한 자세한 내용은 [2026.01.0 릴리스의 새로운 기능](whats-new-2026-01-0.md)을 참조하십시오.

[2026.01.0 릴리스의 업그레이드 지침](upgrade-instructions-2026-01-0.md)에 대해 알아봅니다.

## 작성

- 컨텍스트 메뉴의 MathML 편집 옵션을 사용하여 인라인 MathML 방정식을 업데이트할 때 페이지를 새로 고칠 때까지 업데이트된 값이 반영되지 않습니다. (GUIDES-38198)
- 주제에 재사용 가능 패널에 추가된 재사용 가능한 요소(ID가 있는 요소)가 많이 포함된 경우, 고정된 컨테이너 높이 때문에 일부 요소에 액세스할 수 없습니다. (GUIDES-37220)
- 파일에 대한 상호 참조를 삽입할 때 맵과 주제의 아이콘은 동일합니다.(GUIDES-36662)
- 맵을 편집할 때 작성자 보기에서 `navtitle`에 대해 `topichead`의 특수 기호가 표시되지 않습니다. (GUIDES-35435)
- **새 버전으로 저장** 대화 상자에서 여러 버전 레이블을 항목에 추가할 수 없습니다. (GUIDES-32716)

## 자산 관리

- Assets UI의 버전 내역 패널에서 버전 레이블을 제거할 수 없습니다. (GUIDES-38276)
- 파일 이름에 잘못된 문자가 들어 있는 자산을 업로드할 때 자산이 업로드되지 않고 **다른 사용자가 파일을 잠갔습니다**&#x200B;라는 잘못된 메시지가 표시됩니다. (GUIDES-32680)
- Assets 검색에서 하위 에셋 및 메타데이터 노드(예: 이미지 및 PDF)가 결과에 잘못 포함됩니다. 또한 DITAVAL 필터가 적용된 출력 사전 설정의 경우 조건 사전 설정에서 생성된 내부적으로 생성된 DITAVAL 파일이 검색에서 반환됩니다. (GUIDES-35418)

## 게시

- AEM Sites 출력을 생성할 때 `<ph>` 요소가 있는 키워드 및 주제 제목이 포함된 맵 제목은 게시된 출력에 포함되지 않습니다. (GUIDES-36641)
- 범위가 외부로 설정되지 않은 상태에서 PDF에 대한 링크가 포함된 콘텐츠와 함께 사용자 지정 사전 설정을 사용하여 게시할 경우 프로세스가 완료되지 않습니다. (GUIDES-21708)
- 일괄 활성화를 수행할 때 패키지를 만들면 외부 및 피어 경로를 포함하여 페이지의 `fileReference` 속성 아래에 나열된 모든 경로에 대한 필터가 추가됩니다. (GUIDES-24887)
- 기본 PDF 출력에서 `abbreviated-form` 요소는 지정된 `glossterm` 또는 `glossSurfaceForm` 대신 `glossAcronym`을(를) 표시합니다. (GUIDES-26393)
- 기본 PDF 출력의 경우 이미지에 대한 `<alt>` 요소가 무시되어 접근성을 위해 대체 텍스트가 적용되지 않습니다. (GUIDES-29087)
- 사전 설정을 게시하는 동안 기준선이 있는 맵의 임시 파일을 다운로드할 때 `metadata.xml` 파일이 `versionPath` 대신 `dampath`을(를) 잘못 참조합니다.(GUIDES-29815)
- 인용이 포함된 주제를 만들거나 편집할 때 인용 대화 상자에 작성자 필드가 추가되지 않으면 PDF이 생성되지 않습니다. (GUIDES-37934)
- 참조되는 CSS가 없어도 CSS 파일(`rhdefault.css`)이 PDF 템플릿에 잘못 적용되어 CSS 파일 오류 로그가 누락됩니다.(GUIDES-31752)

## Platform

- 주제나 맵을 저장하려고 할 때 작업이 간헐적으로 실패하고 **파일 저장 실패** 오류가 발생할 수 있습니다. 특히 백그라운드에서 실행되는 집중 자산 처리 작업 또는 번역 워크플로 중에 이러한 오류가 발생할 수 있습니다. (GUIDES-37837)
- 항목 또는 맵 내에서 DAM 콘텐츠에 대한 참조에 `scope="external"`을(를) 사용하면 자산의 상대 경로가 GUID로 대체됩니다. (GUIDES-38783)

## 보고서

- 끊김 목록 보고서에 현재 맵의 범위 내에서 올바르게 확인된 외부 링크, 올바른 `keyrefs` 및 키워드가 잘못 포함되어 있습니다. (GUIDES-27774)

## 알려진 문제

Adobe은 2026.01.0 릴리스에 대해 다음과 같은 알려진 문제를 확인했습니다.

- 검토 중인 항목이 진행 중인 검토 작업에서 제거되면 해당 문서 상태는 더 이상 검토 작업에 포함되지 않더라도 계속 **검토 중** 상태로 유지됩니다. (GUIDES-38709)<br>**해결 방법**: 주제의 문서 상태를 **검토 중**&#x200B;에서 속성 페이지 또는 파일 속성 패널에서 적절한 상태로 변경합니다.
- **찾기 및 바꾸기**&#x200B;를 사용하여 검색을 수행할 때 검색 결과에서 파일을 열고 닫은 다음 나열된 결과를 선택하여 다시 열려고 하면 파일을 다시 열 수 없습니다. (GUIDES-39050)<br>**해결 방법**: 검색 결과에서 다른 파일을 먼저 연 다음 목록에서 이전에 닫은 파일을 다시 열어 문제를 해결하십시오.
- DB 서버와 함께 Guides를 사용할 때 자체 참조가 포함된 컨텐츠의 경우 항목 목록 보고서에 각 자체 참조에 대해 잘못된 항목이 표시되므로 파일 수가 부정확합니다. (GUIDES-39420)
