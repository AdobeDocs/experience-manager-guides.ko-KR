# 2026.03.0 릴리스의 문제가 해결되었습니다.

이 문서에서는 Adobe Experience Manager Guides as a Cloud Service의 2026.03.0 릴리스의 다양한 영역에서 수정된 버그를 다룹니다.

새로운 기능 및 개선 사항에 대한 자세한 내용은 [2026.03.0 릴리스의 새로운 기능](whats-new-2026-03-0.md)을 참조하십시오.

[2026.03.0 릴리스의 업그레이드 지침](upgrade-instructions-2026-03-0.md)에 대해 알아봅니다.

## 작성

- Schematron(`*.sch`) 파일을 편집하고 찾기 및 바꾸기 기능을 사용하면 찾기 및 바꾸기 패널이 맨 아래에 일부 화면 밖에 표시되어 입력 필드 및 컨트롤에 액세스할 수 없습니다. (GUIDES-38412)

## 게시

- 조건부 사전 설정이 다른 여러 맵에서 동일한 주제가 재사용되는 경우 최신 맵을 Salesforce에 게시하면 주제 콘텐츠가 덮어쓰므로 이전에 게시된 맵의 사용자에게 잘못된 데이터가 표시됩니다. (GUIDES-37806)
- 조건부 처리 또는 특정 중첩된 맵이 포함된 맵에 대해 기본 PDF을 게시할 때 맵에 정의된 `dc:title`이(가) PDF 표지에 표시되지 않아서 표지 제목이 누락됩니다. (GUIDES-37733)
- `map_title` 변수가 출력 경로에 있는 경우 맵에 대한 AEM 사이트 출력(복합 구성 요소 매핑 사용)을 생성하지 못하고 오류가 발생합니다. (GUIDES-36968)
- 기본 PDF 출력 사전 설정에서 후행 슬래시가 있는 출력 경로를 지정하면 UI에서 후행 슬래시를 자동으로 추가하여 특정 시나리오에서 PDF 업로드가 실패하는 이중 슬래시 경로를 생성합니다. (GUIDES-34932)
- 빠른 게시 또는 게시 관리를 통해 DITA 콘텐츠에서 생성된 AEM Sites 페이지를 게시하면 관련 DITA 에셋이 실수로 게시됩니다. (GUIDES-27967)
- 기존 구성 요소 매핑을 사용하여 AEM Sites에 맵을 게시할 때 다른 맵에서 항목(예: 단락)의 하위 요소를 타겟팅하는 `xrefs`을(를) 사용하는 상호 참조(`scope = peer`)는 특정 요소 ID로 확인되지 않고 대신 상위 항목으로만 확인되므로 페이지가 의도한 섹션으로 스크롤되지 않고 항목 시작 시 로드됩니다. (GUIDES-21802)


## 번역

- 처음에 특정 버전(예: `/en/` 아래)의 언어별 자산으로 관리된 이미지를 업데이트된 버전의 전역 폴더로 이동하고 기준 요소 내보내기가 수행되면 새 기준 요소가 해당 이미지의 오래된 언어별 버전을 계속 참조하여 기준 요소 내보내기가 실패합니다. (GUIDES-39394)
- Experience Manager Guides 외부에서 만든 번역 프로젝트를 처리할 때 *`fmTarget`속성을 찾을 수 없다는 오류 로그 메시지가 여러 개 생성됩니다*. (GUIDES-37804)

## 베이스라인

- 동적 기준선을 만들면 여러 개의 동시 API 요청으로 인해 편집기가 응답하지 않아 다른 모든 작업이 중지되는 경우가 있습니다. (GUIDES-39054)

## 검토

- 검토 작업에 사용자를 할당할 때 드롭다운에 선택한 프로젝트와 관련된 사용자만 나열하는 대신 모든 사용자가 나열되므로 사용자 옵션이 유효하지 않습니다. (GUIDES-37781)

## 보고서

- 맵에 대한 보고서를 여는 동안 필터 패널 로드가 지연됩니다(GUIDES-39385)

## 알려진 문제

Adobe은 2026.03.0 릴리스에 대해 다음과 같은 알려진 문제를 확인했습니다.

- 중복 ServiceNow 기술 자료 사전 설정을 만들 때 빈 화면이 로드됩니다. (GUIDES-42732)
- 문서 상태 검색을 위한 API가 일부 파일에 대해 null 응답을 반환하여 잘못된 문서 상태가 UI에 표시됩니다. (GUIDES-42561)
- 맵 대시보드의 탭 이름을 기반으로 한 UI 사용자 지정은 적용되지 않습니다. (GUIDES-42285)
- 파일을 편집기와 검색 패널에서 모두 열면 [탐색기] 패널에서 해당 파일을 삭제하면 파일이 제거되고 [탐색기] 목록이 새로 고침되지만 페이지를 새로 고치면 [검색] 패널에 파일이 계속 표시됩니다. (GUIDES-41935)
- 활성 검토 작업을 업데이트하는 동안 이미 검토에 포함된 주제를 제거한 다음 **업데이트**&#x200B;를 클릭하지 않고 다시 추가하면 검토자 탭의 검토자 정보가 손실됩니다.   (GUIDES-38774)
- 주제가 북맵 태그(앞면, 장, 부분 또는 뒷면) 또는 순환 콘텐츠의 일부에 있는 경우 [미리 보기] 모드에서 주제를 선택하면 맵 보기에서 주제가 강조 표시되지 않습니다. (GUIDES-42416)
- Assets UI에서 두 개 이상의 파일 또는 폴더를 선택한 경우 첫 번째 시도에서 **이동** 단추가 활성화되지 않습니다. (GUIDES-42721) <br> **해결 방법**: 파일 또는 폴더를 두 개 이상 선택한 후 대상 폴더를 선택하기 전에 몇 초 동안 기다리십시오.
- 편집기에서 미리 보기 모드가 열려 있는 동안 편집기에서 **사용자 환경 설정**(으)로 이동하여 루트 맵을 업데이트하면 편집기로 돌아가면 맵 미리 보기가 빈 화면으로 로드됩니다. (GUIDES-42412) <br> **해결 방법**: 미리 보기 모드에서 사용할 수 있는 **새로 고침** 아이콘을 사용하여 미리 보기를 새로 고치면 문제가 해결됩니다.
- 기존 템플릿의 이름을 바꾸면 페이지를 수동으로 새로 고칠 때까지 **출력 템플릿** 패널의 이름이 업데이트되지 않습니다. (GUIDES-42528)<br> **해결 방법**: [출력 템플릿] 패널에서 업데이트된 템플릿 이름을 보려면 브라우저를 새로 고치십시오.
