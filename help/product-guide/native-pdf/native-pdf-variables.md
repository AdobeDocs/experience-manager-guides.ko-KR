# PDF 출력의 변수

변수은 재사용 가능한 정보 조각으로 사용 되는 데이터의 이름/값 쌍입니다. 이렇게 하면 컨텐츠 이식 가능 하 고 쉽게 업데이트할 수 있습니다. 변수 또는 값을 수정 하면 해당 변수 또는 값의 모든 발생 횟수 업데이트 됩니다.



## 새 변수 만들기

변수를 만들려면 다음 단계를 수행하십시오.

![새 변수 만들기](assets/add-variable-default.png){width="800" align="left"}

*변수를 만들고 값을 정의합니다.*


1. 웹 편집기에서 **출력** 탭.
1. 선택 **변수** <img alt= "변수 아이콘" src="./assets/variables-icon.svg" width="25"> 왼쪽 패널에서
1. 선택 **편집** <img alt= "연필 아이콘 편집" src="./assets/edit_pencil_icon.svg" width="25"> 을(를) 열려면 **변수** 편집자.
변수는 알파벳 순으로 나열 됩니다.
1. 이름 열에 **변수 이름을 입력 하 고 값** 열에 **값을 입력** 합니다.
   >[!TIP]
   >
   >모든 HTML 콘텐츠를 변수 값으로 사용하여 특정 서식으로 변수 값을 표시할 수 있습니다. 예를 들어 다음을 추가할 수 있습니다 `<b>` 변수 값에 태깅하여 값을 표시합니다. **Experience Manager 가이드** 굵게. 저장소의 이미지를 값으로 추가할 수도 있습니다.

1. 선택 **변수 추가** <img alt= "추가 아이콘" src="./assets/add-icon.svg" width="25"> 을 클릭하여 새 변수를 추가합니다. 기존 변수와 이름이 같은 변수는 만들 수 없습니다. 오류가 표시됩니다.

   >[!NOTE]
   >
   >을 선택하지 않는 경우 **변수 추가** <img alt= "추가 아이콘" src="./assets/add-icon.svg" width="25">: 변수가 만들어지지 않고 목록에 추가됩니다.

이러한 방식으로 기본값을 사용하여 변수를 만들 수 있습니다. 예:
* ProductName: Experience Manager 가이드
* 버전 번호: 2300
* 릴리스 날짜: 2023/01/01


### 변수 편집

다음 두 가지 방법으로 변수를 편집할 수 있습니다.

**왼쪽에 있는 변수 패널에서**

1. 에서 변수 선택 **변수** 패널.
1. 변수를 마우스로 가리키면 **옵션** 메뉴를 선택한 다음 **편집** 옵션을 선택합니다.
1. 다음에서 **변수 편집** 대화 상자에서 선택한 변수의 기본값을 편집할 수 있습니다.
1. **완료**&#x200B;를 클릭합니다.

**변수 편집기에서**

1. 선택 **변수** <img alt= "변수 아이콘" src="./assets/variables-icon.svg" width="25"> 왼쪽 패널에서
1. 선택 **편집** <img alt= "연필 아이콘 편집" src="./assets/edit_pencil_icon.svg" width="25">변수 **편집기를 열려면** .

1. **변수** 편집기에서 선택한 변수의 값을 편집할 수 있습니다.

변수 편집기에서 **수행한 변경 사항을 저장 하 여 왼쪽의 [변수** ] 패널에서 **확인 해야** 합니다.

>[!NOTE]
>
> 변수 값을 편집 하는 경우 Adobe Experience Manager 가이드가 해당 되 면 언제 든 지 모든 참조를 업데이트 합니다.

### 변수 Search 및 미리 보기

변수 값을 검색하고 미리 볼 수 있습니다. 의 검색 상자에 문자열을 입력합니다. **변수** 패널. 변수 이름과 해당 값을 기준으로 모두 검색합니다.
다음 두 가지 방법으로 변수를 미리 볼 수 있습니다.

변수 미리보기에는 기본값이 표시됩니다. 예를 들어 ProductName 변수의 기본값을 &quot;Adobe Experience Manager Guides&quot;로 정의한 경우 미리보기에 이 값이 표시됩니다.

**왼쪽에 있는 변수 패널에서**


1. 에서 변수 선택 **변수** 패널.
1. 변수를 마우스로 가리키면 **옵션** 메뉴를 선택한 다음 **미리 보기** 옵션을 선택합니다.

![변수 패널에서 변수 미리 보기](assets/variables-panel-preview-default.png){width="550" align="left"}

*변수에 대한 기본값을 미리 봅니다.*

**변수 편집기에서**

1. 목록의 변수 위로 마우스를 가져가면 **옵션** 메뉴 아래의 제품에서 사용할 수 있습니다.
1. 선택 **미리 보기**.




### 변수 복제

변수를 복제 하 고 요구 사항에 따라 값을 수정할 수 있습니다.


1. 목록의 변수 위로 마우스를 가져가면 옵션 **메뉴를 볼** 수 있습니다.
1. 복제 **를 선택** 합니다.

변수의 기본 이름은 입니다. `<selected variable name>` (예: &quot;샘플&quot;). 필요에 따라 이름을 변경할 수 있습니다.

### 변수 삭제

다음 두 가지 방법으로 변수를 삭제할 수 있습니다.

**왼쪽에 있는 변수 패널에서**

1. 에서 변수 선택 **변수** 패널.
1. 변수를 마우스로 가리키면 **옵션** 메뉴를 선택한 다음 **삭제** 옵션을 선택합니다.

**변수 편집기에서**

1. 목록의 변수 위로 마우스를 가져가면 **옵션** 메뉴 아래의 제품에서 사용할 수 있습니다.
1. 선택 **삭제** 옵션을 선택합니다.

변수는 모든 변수 세트에서 삭제됩니다.

## 출력 사전 설정에 대한 변수 세트

Adobe Experience Manager 안내선은 변수 대체 값을 할당할 수 있는 변수 세트도 지원 합니다. 예를 들어 회사는 A와 B 라는 두 개의 제품을 판매할 수 있습니다. 여기에는 각각 다른 사양이 있습니다. 이러한 사양에는 제품 이름, 버전 번호 및 릴리스 날짜가 포함 될 수 있습니다. 브랜딩에는 다른 차이점이 있을 수 있습니다. 변수 세트를 사용하여 변수에 대해 다른 값 세트를 정의합니다. 출력을 생성할 때 적절한 변수 세트를 선택하고 필요한 출력을 생성합니다.

### 변수 집합 구성

변수를 추가하기 전에 변수 세트를 구성해야 합니다.



1. 선택 **설정** <img alt= "설정 아이콘" src="./assets/settings-icon.svg" width="25"> 을(를) 열려면 **변수 집합 구성** 대화 상자.
   ![변수 집합 구성](assets/configure-variable-set.png){width="550" align="left"}
1. 에 변수 세트 이름을 입력합니다. **이름** 열.
1. 선택 **변수 추가** <img alt= "추가 아이콘" src="./assets/add-icon.svg" width="25"> 새 변수 세트를 추가합니다. 변수 세트는 알파벳순으로 나열됩니다.
1. 다음을 선택할 수 있습니다. **삭제** 변수 세트를 제거합니다.

### 변수 집합 작업

모든 변수 집합은 동일한 변수를 가지지만 다른 값을 가질 수 있습니다.

특정 변수 세트의 값을 보고, 편집하고, 미리 볼 수 있습니다. 다음에서 변수 세트를 선택합니다. **변수 집합** 드롭다운입니다. 선택한 변수 세트에 따라 값이 표시됩니다.
특정 변수 세트의 변수 값을 편집하면 기본값이 무시되고 선택한 변수 세트의 값이 변경됩니다.
예를 들어 변수 세트에 대해 다음 값을 설정할 수 있습니다. *Adobe-세트1* 및 *Adobe 집합2* .


**변수 세트 1**: *Adobe-세트1*

* 제품 이름: 제품 A
* 버전 번호: 2311
* 릴리스 날짜: 2023/11/02


**변수 집합 2**: *Adobe 집합2*

* 제품 이름: 제품 B
* 버전 번호: 2310
* 릴리스 날짜: 2023/09/07

모든 새 변수는 모든 변수 세트에 추가됩니다. 변수를 삭제하거나 복제하면 모든 변수 세트에 대해 업데이트됩니다.

변수 세트의 값을 미리 볼 수도 있습니다.
예를 들어 변수 세트의 경우 *Adobe-집합1*, ProductName 변수의 값을 &quot;ProductA&quot;로 정의한 다음 변수 편집기의 미리보기에 이 값을 표시합니다.

![변수 편집기에서 변수 미리 보기](assets/variables-editor-preview.png){width="550" align="left"}

*선택한 변수 세트에서 정의한 값을 미리 봅니다.*

### 변수 값 재설정

값을 편집한 경우 변수를 기본값으로 재설정할 수도 있습니다.
재설정 <img alt= "재설정 아이콘" src="./assets/application-variable-revert.svg" width="25"> 변경된 값이 있는 변수에 대해 표시됩니다.
예를 들어 ProductName 변수의 값을 기본값 Experience Manager Guides로 재설정할 수 있습니다.

## 기본 PDF 템플릿에서 변수 사용

제품 문서의 출력을 생성하는 동안 변수를 추가하여 휴대하고 쉽게 업데이트할 수 있습니다. 문서의 여러 페이지에 표시되는 페이지 레이아웃 내에 이러한 변수를 삽입할 수 있습니다. 예를 들어 페이지 레이아웃의 머리글 영역(또는 바닥글이나 본문 같은 다른 부분)에 나타나는 변수 ProductName을 추가할 수 있습니다.



헤더 영역에 ProductName과 같은 변수를 삽입하려면 다음 단계를 수행하십시오.
1. 편집에 필요한 페이지 레이아웃을 엽니다.

   >[!NOTE]
   >
   > 보기 [페이지 레이아웃 사용자 지정](../native-pdf/components-pdf-template.md#customize-a-page-layout-customize-page-layout) 사용자 지정 또는 편집을 위한 페이지 레이아웃을 열기 위한 섹션입니다.

1. 변수를 삽입하도록 활성화하려면 헤더를 선택합니다.

1. 다음 두 가지 방법으로 변수를 삽입할 수 있습니다.

   **왼쪽의 [변수] 패널에서**

   * [변수 **] 패널에서** 변수를 드래그 하 여 머리글 영역에 놓습니다.

   **도구 모음에서**

   1. 선택 **변수/필드 삽입** <img alt= "변수 아이콘" src="./assets/variables-icon.svg" width="25">
   1. 다음에서 **변수** 대화 상자에서 헤더 영역에 삽입할 변수의 이름을 선택합니다.
   1. 텍스트 상자에 검색 문자열을 입력할 수도 있습니다. 주어진 문자열을 포함하는 변수 이름은 필터링되고 목록에 표시됩니다. 선택한 변수가 헤더 영역에 삽입됩니다. 변수의 기본값을 볼 수 있습니다.
   1. 변수를 바꾸려면 변수 값을 두 번 클릭하고 **변수** 대화 상자. 변수가 대체됩니다.


## 변수를 사용하여 PDF 출력 생성

다양한 변수의 값을 사용하여 PDF 출력을 생성할 수 있습니다. 레이아웃을 생성하기 전에 출력 사전 설정에서 변수 세트를 선택합니다 **변수 세트** 드롭다운 목록에서 해당 값을 선택합니다.

![변수 세트 드롭다운](assets/output-preset-variable-dropdown.png){width="550" align="left"}

*PDF 출력을 생성 하는 데 사용할 [출력 사전 설정]의 드롭다운에서 변수 집합을 선택 합니다.*

>[!NOTE]
>
> 드롭다운 목록에서 (기본값)를 선택 하 여 모든 변수에 대 한 기본값을 게시 수도 있습니다.

선택한 변수 세트에 따라 변수 세트에 정의된 변수 값에 해당하는 출력을 가져옵니다. 예를 들어 변수 세트를 선택하는 경우 *Adobe-세트1*&#x200B;를 입력하면 이 세트에 정의된 대로 변수 값이 표시됩니다.


<img src="assets/variable-pdf-output.png" alt="변수가 있는 PDF 출력" width="500" border="2px">

*페이지 레이아웃의 변수를 사용하여 PDF 출력을 생성합니다.*

필요할 때마다 변수 세트의 값을 빠르게 업데이트하고 출력을 재생성할 수도 있습니다. 예를 들어 버전에 대한 세부 정보를 업데이트해야 하는 경우 VersionNumber 변수에서 버전 값을 업데이트하고 출력을 재생성할 수 있습니다.
