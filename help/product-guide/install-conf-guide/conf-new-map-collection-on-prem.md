# 온-프레미스에 대한 새 맵 컬렉션 구성

다음 단계에서는 온-프레미스 환경에서 새 맵 컬렉션을 활성화하는 방법을 설명합니다.

1. Adobe Experience Manager 웹 콘솔 구성 페이지를 엽니다.

   구성 페이지에 액세스하기 위한 기본 URL은 다음과 같습니다.

   ```http
   http://<server name>:<port>/system/console/configMgr
   ```

1. **com.adobe.fmdita.config.ConfigManager** 번들을 검색하고 선택합니다.

1. **새 맵 컬렉션 사용** 설정을 사용합니다(enable.new.map.collections).

   ![](assets/new-map-collection.png){width="800"}

1. **저장**&#x200B;을 선택합니다.
