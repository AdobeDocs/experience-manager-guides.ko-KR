# 편집기 사용자 지정 {#id181AI0003PN}

AEM Guides에는 작성자가 DITA 문서를 만들고 편집할 수 있는 강력한 편집기가 번들로 제공됩니다. 편집기 도구 모음을 사용자 지정하여 도구 모음에서 액세스할 수 있는 기능을 추가하거나 제거할 수 있습니다. 또한 파일의 자동 저장을 구성하고, 요소에 삽입되면 해당 요소에 대한 ID를 생성하는 등의 작업을 수행할 수 있습니다.
