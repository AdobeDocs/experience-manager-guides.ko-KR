# 면책 조항 {#id2263D0P0TYU}

*Adobe Experience Manager Guides*&#x200B;은(는) 이전에 *Adobe Experience Manager용 XML Documentation* 브랜드로 지정되었습니다. 설명서 내의 특정 참조는 이전 브랜딩을 계속 참조할 수 있지만 현재 제공에 여전히 적용할 수 있습니다.
