# 산소를 사용한 작업 {#id221BH0V0UE8}

Oxygen XML Author에는 사용자 친화적인 작성 기능이 포함되어 있으며 사용하기 쉬운 인터페이스를 제공합니다.

동일한 소스를 사용하여 PDF, ePub, HTML 및 기타 다양한 형식으로 출력을 생성할 수 있습니다.

*고객 지원 팀에 문의하여 환경에서 이 기능을 활성화하십시오. 기본 지원의 일부로 사용할 수 없습니다.*
