# 편집기 소개 {#id181OD0QE05Z}

Experience Manager Guides에는 구조화된 문서를 만들고 편집할 수 있는 사용하기 쉬운 편집기가 포함되어 있습니다. 편집기는 작성기에서 DITA 구조의 모든 복잡성을 숨깁니다. 편집기는 사용자가 일반적으로 문서 내에서 작업하는 데 필요한 DITA 요소 목록을 제공합니다.

## 새 편집기

>[!NOTE]
>
> 새로운 편집기는 2026.05.0 릴리스부터 Experience Manager Guides as a Cloud Service에서 사용할 수 있습니다. 이 기능을 활성화하려면 고객 지원 팀에 문의하십시오.

새 편집기는 성능, 유용성 및 UI/UX에서 중점적으로 개선되어 작성 환경의 상당한 진화를 제공합니다. DITA 맵 및 DITA 주제용으로 특별히 설계된 이 솔루션을 사용하면 DITA가 아닌 컨텐츠에는 영향을 주지 않고 완벽하게 호환되는 동시에 크고 복잡하며 구조가 높은 컨텐츠를 효율적으로 작성할 수 있습니다.

### 주요 기능

- 대용량 파일의 성능 향상: 실행 취소/다시 실행 및 더티 마커를 지원하여 대형 DITA 파일의 응답성을 향상하므로 성능 저하 없이 안정적인 변경 내용 추적을 수행할 수 있습니다.
- 현대화 테이블 작성: 드래그 앤 드롭 재정렬, 상황별 도구 모음, 대량 행 또는 열 삽입을 통해 보다 직관적인 테이블 편집 환경을 제공하여 보다 빠르고 정확한 테이블 관리를 수행할 수 있습니다.
- 향상된 작성자 모드: 작성자 모드에서 직접 XML 주석 및 속성을 구성 가능한 가시성으로 제공하여 구조화된 컨텐츠로 작업하는 동안 Source 보기로 전환할 필요성을 줄여줍니다.
- 고급 편집 모드 및 도구: 주제 및 빠른 삽입 메뉴에 대한 나란히 작성자 및 Source 모드를 도입하여 최소한의 컨텍스트 전환으로 더 빠른 구조화된 작성을 가능하게 합니다.
- UI 및 UX 개선: 작성자 및 미리보기 모드에서 어두운 테마와 풍부한 기본 CSS를 통해 가독성과 시각적 편안함을 개선하여 보다 현대적인 편집 인터페이스를 제공합니다.
- 중앙 편집기 설정: 태그, 특성, XML 주석, 줄바꿈 없는 공백 및 편집기의 빠른 삽입 메뉴에 대한 제어를 포함하여 작성 환경을 개인화할 수 있는 통합 설정 패널을 제공합니다.

### 새 편집기의 제한 사항

새 편집기는 현재 다음 기능을 지원하지 않습니다. 이러한 기능은 향후 릴리스에 도입될 예정입니다.

- 변경 내용 추적이 활성화된 작성(기존의 추적된 변경 내용은 모두 일반 컨텐츠로 유지 및 처리됨)
- 새 인용(콘텐츠의 기존 인용 횟수는 보존됨)
- 작성자 보기의 이미지 크기 조정기
- 데이터 소스의 데이터 사용(예: JIRA, SQL 데이터베이스, PIM 또는 PLM)

>[!NOTE]
>
>DITA 맵에 대한 레이아웃 보기는 새 편집기에서 지원되지 않습니다.


## 이전 편집기

Experience Manager Guides의 이전 편집기는 구조화된 컨텐츠를 위한 간단하고 안정적인 작성 환경을 제공하여 DITA 1.2, 1.3, 경량 DITA 및 특수화된 DITA와 같은 지원되는 표준을 준수하면서도 DITA의 복잡성을 추상화합니다. 이는 일반적인 블록 및 인라인 요소로 작업하는 데 필요한 도구를 제공하므로 구조화된 프레임워크 내에서 일관되고 통제되는 콘텐츠 생성에 적합합니다.
