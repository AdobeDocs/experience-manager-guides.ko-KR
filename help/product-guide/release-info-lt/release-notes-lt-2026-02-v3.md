# 2026년 2월 제품 교육 및 학습 콘텐츠 릴리스

이 릴리스 노트에서는 제품 교육 및 학습 컨텐츠의 2026년 2월 릴리스에서 해결된 새로운 기능 개선 사항 및 문제를 다룹니다.

## 새로운 기능 개선 사항

제품 교육 및 학습 컨텐츠의 2026년 2월 릴리스에는 다음 기능이 도입되었습니다.

- **자막 지원**: 이제 **파일 속성**&#x200B;에서 새로운 **자막 추가** 옵션을 사용하여 학습 콘텐츠에 자막을 추가할 수 있습니다. 이를 통해 명확성이 향상되고 과정 콘텐츠 전반에서 검색 기능이 향상됩니다.

  자세한 내용은 [학습 콘텐츠에 제목 및 부제목 추가](../learning-content/lc-basic-blocks.md#add-title-and-subtitle-to-learning-content)를 참조하세요.

  ![](assets/add-subtitles.png)

- **최종 부정 채점 활성화 또는 비활성화**: 퀴즈 속성을 구성하는 동안 **부정 최종 점수 허용** 옵션을 사용하여 부정 채점을 제어할 수 있습니다. 활성화되면 학습자는 부정 마크가 적용되더라도 최소 최종 점수 0을 받습니다. 이는 점수가 영점 이하로 떨어지지 않도록 하여 학습자에게 동기를 부여합니다.

  [퀴즈 속성](../learning-content/quiz-properties.md)에 대해 자세히 알아보세요.

  ![](assets/negative-scores-lc.png)

- **마우스 오른쪽 단추로 클릭하여 위젯을 삭제**: 이제 퀴즈 문제를 삭제하는 것 외에도 **마우스 오른쪽 단추 클릭 > 항목 삭제**&#x200B;를 사용하여 아코디언, 플립카드 및 탭과 같은 위젯을 삭제할 수 있습니다. 이 개선 사항은 기존 *질문 삭제* 기능을 위젯으로 확장하므로 클릭수를 줄이고 탐색을 최소화하면서 질문을 제거할 수 있습니다.

  [대화형 위젯 사용](../learning-content/lc-widgets.md)에 대해 자세히 알아보세요.

  ![](assets/delete-widget-items.png)
- **답변 선택 항목 고정**: 이제 SCORM 출력 생성 중 응답이 임의 지정되는 경우에도 특정 답변 선택 항목을 고정할 수 있으므로 위치가 변경되지 않습니다. 이는 *위의 모든 항목* 또는 *위의 항목 없음*&#x200B;과 같은 옵션에 특히 유용합니다.

  ![](assets/pin-question.png)
- **짧은 답변 유형**: 짧은 답변 질문 유형을 사용하면 학습자가 사전 정의된 옵션을 선택하는 대신 간단한 영숫자 답변을 사용하여 응답할 수 있습니다. 이러한 질문유형은 학습자가 자신의 이해력을 적극적으로 회상하고 자신의 말로 표현하도록 유도하여 학습자에게 평가를 더욱 매력적인 문항으로 만든다.

  [질문 유형](../learning-content/quiz-insert-questions.md#question-types)에 대해 자세히 알아보세요.


  ![](assets/short-answer.png)
- **퀴즈 질문에 대한 순차적 시도**: 이제 **학습자가 진행하려면 모든 질문을 시도해야 합니다** SCORM 출력 사전 설정 옵션을 사용하여 SCORM 출력에 대해 순차적 퀴즈 시도를 적용할 수 있습니다. 활성화되면 학습자는 다음 질문으로 이동하기 전에 각 질문에 답해야 하며 현재 질문이 완료될 때까지 탐색이 제한됩니다. 이를 통해 단계별 평가 흐름과 일관된 학습 경험이 보장된다.

  자세한 내용은 [SCORM 출력 사전 설정 구성](../learning-content/config-scorm-preset.md)을 참조하십시오.

  ![](assets/scorm-general-tab-v3.png)

## 해결된 문제

제품 교육 및 학습 컨텐츠의 2026년 2월 릴리스에서 다음 문제가 수정되었습니다.

- SCORM 출력을 게시하고 ALM에 배포할 때 L2 퀴즈 보고서에 여러 번의 시도와 무작위 질문 뱅크 선택을 사용하는 퀴즈의 잘못된 총점과 최대 점수가 표시됩니다. (GUIDES-38855)
- 클라우드 서버에서 과정이 생성되면 `coralui3.css` 스타일시트로 인해 저작권 바닥글 아래에 의도하지 않은 공백이 표시되어 레이아웃 불일치가 발생합니다. (GUIDES-38853)
- 키보드를 사용하여 아코디언이 있는 학습 과정을 탐색할 때 + 기호 또는 탭 제목이 강조 표시되지 않아서 활성 요소가 시각적으로 식별되지 않습니다. (GUIDES-38852)
- SCORM 카본 템플릿 또는 기본 템플릿을 사용하여 생성된 강의의 경우, 가로 모드로 모바일 장치에서 액세스할 때 목차(강의 메뉴)에 탐색을 방해하는 모듈 링크가 표시되지 않습니다. (GUIDES-38851)
- Experience Manager Guides에서 강의에 대한 계층을 복제할 때 개체 수준의 추가가 지원되지 않으므로 학습 개체를 만들려면 먼저 학습 그룹을 만들어야 합니다. (GUIDES-38849)
- 키보드를 사용하여 다음 질문 유형 일치에서 드롭다운 옵션에 액세스하려고 하면 옵션이 탐색을 방해하는 탭 또는 화살표 키에 응답하지 않으므로 실패합니다. (GUIDES-38985)
- 제목 스타일 사전 설정을 적용하면 글꼴 색상이 흰색으로 변경되어 선택한 텍스트가 사라지고 텍스트를 선택할 수 없게 되고 표시되지 않게 됩니다. (GUIDES-39981)
- Mozilla Firefox에서 Experience Manager Guides을 사용할 때 뒤집기 카드는 뒤집기 후 앞면 텍스트를 뒷면에 반대로 표시합니다. (GUIDES-39983)
- 과정 왼쪽 창에서 목차(TOC)를 클릭하면 퀴즈에 실패했더라도 과정 완료 상태가 계속 표시됩니다. (GUIDES-40398)
- ALM에서 퀴즈의 다음 질문 유형과 일치시키려고 하면 선택한 옵션이 보고서에 표시되지 않습니다. (GUIDES-38640)
- PDF 출력을 생성할 때 적용된 작성 스타일이 유지되지 않으므로 디자인이 일치하지 않습니다.(GUIDES-38642)
